# Swardhun - Music Streaming App

A beautiful music streaming application built with Android Kotlin and Jetpack Compose, featuring JioSaavn integration.

## Features

- Stream millions of songs from JioSaavn
- Beautiful glassmorphism UI inspired by Apple Music & Spotify
- Download songs for offline listening
- Like and save your favorite songs
- Shuffle and repeat modes
- Background playback with notification controls
- Auto-update notifications when new version is available
- Multi-language music support (Hindi, English, Punjabi, Bhojpuri, and more)

## Installation

### Android
1. Download the APK file from releases
2. Enable "Install from unknown sources" in settings
3. Install the APK
4. Enjoy your music!

### Building from Source

\`\`\`bash
# Clone the repository
git clone https://github.com/AryanVBW/Swardhun.git
cd Swardhun

# Build debug APK
./gradlew assembleDebug

# Build release APK
./gradlew assembleRelease
\`\`\`

## Tech Stack

- **Language**: Kotlin
- **UI**: Jetpack Compose with Material 3
- **Architecture**: MVVM
- **Audio Playback**: ExoPlayer / Media3
- **Database**: Room
- **Network**: OkHttp + Kotlin Coroutines
- **Image Loading**: Coil
